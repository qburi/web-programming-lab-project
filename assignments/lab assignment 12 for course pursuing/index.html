Ordered List
An ordered list is a collection of items where the sequence is important. The order suggests priority, hierarchy, or a step-by-step process
Unordered List
An unordered list is a collection of related items that have no special order or sequence. Changing the arrangement of the items would not change the meaning of the list.
Description List
A Description List is a flexible list structure designed to group terms with their corresponding details.
